## 📌 Sobre o Projeto

Este projeto é uma **implementação básica de uma API** usando Python e Flask. O objetivo é demonstrar uma estrutura simples de API com respostas em JSON e algumas funcionalidades básicas, servindo como exercício introdutório para entender os conceitos de desenvolvimento de APIs RESTful.

> 🎯 Foco em backend, endpoints REST e retorno de dados em JSON.

## 🚀 Como Executar

### Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- ✅ [Python](https://www.python.org) (versão 3.7+)
- ✅ pip (gerenciador de pacotes do Python)

### Passo a Passo

```bash
# 1. Clone o repositório
git clone https://github.com/RDEsley/API-Simples-Python.git

# 2. Acesse a pasta do projeto
cd API-Simples-Python

# 3. Instale o Flask
pip install flask

# 4. Execute a aplicação
python app.py
```

A API estará disponível em `http://localhost:5000`

### Endpoints Disponíveis

Acesse no navegador ou via ferramenta de API (Postman, Insomnia, etc.):

- http://localhost:5000/api/produtos/
- http://localhost:5000/api/pedidos/
- http://localhost:5000/api/pagamentos/
- http://localhost:5000/api/estoque/
- http://localhost:5000/api/entregas/
- http://localhost:5000/api/clientes/
